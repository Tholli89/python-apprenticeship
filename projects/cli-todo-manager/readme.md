# CLI Todo Manager

A small command-line todo application built to practice Python fundamentals and early project organization.

## Purpose

This project focuses on building confidence with core Python syntax and control flow while organizing code into understandable pieces.

## Features

- Add tasks
- View tasks
- Update task state
- Practice command-line program flow
- Refactor logic into cleaner functions

## Concepts practiced

- Variables and data types
- Lists and dictionaries
- Conditionals and loops
- Functions
- Program organization
- Refactoring

## Why this project matters

This project served as an early foundation project. It introduced the repeated pattern of working with structured records and separating program behavior into smaller functions.

## How to run

From this project folder:

```bash
python main.py
```
